Dotfiles
---

Set zsh as your login shell:
```sh
chsh -s $(which zsh)
```

## install

```sh
git clone --recursive https://github.com/Kei18/dotfiles $HOME/dotfiles && cd dotfiles && bash ./install.sh
```
